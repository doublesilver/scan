[D+1 완료 작업]

1. 전달해주신 데이터 분석 완료
- 쿠팡 SKU 데이터 11,821건 (17개 컬럼) 구조 파악 및 DB 적재 확인
- 바코드-이미지 매핑 데이터 11,821건 파싱 완료
- 기존에 말씀하셨던 "쿠팡 헤더가 바뀌면 인식 실패" 문제 → 컬럼명 자동 인식 로직으로 대응했습니다

2. 서버 API 구조 구성
- 바코드 조회, 상품 검색, 이미지 표시 API
- 개발 환경에서 동작 확인 완료 (조회 응답 약 0.002초)
- NAS 이미지 연동 구조 준비 (접속 정보 수신 후 설정값만 변경하면 연결됩니다)

3. PDA 앱 코드 작성
- 말씀하신 대로 상품 사진이 가장 크게, 가장 먼저 보이도록 화면 설계했습니다
- 현재는 기본 틀만 잡아둔 상태이며, 대표님께서 화면 스케치 주시면 그에 맞춰 UI를 조정하겠습니다
- DataWedge 스캐너 연동 코드 포함 (PDA 수령 후 세팅 예정)

4. 말씀하신 부분 반영 사항
- "발주 담당자" 등 불필요 필드 → DB 구조에서 제외, 향후 도매처 연락처/URL 등 필요하신 필드로 교체 가능하도록 확장 구조로 설계
- 상품 이미지 크게 표시 + 대략적인 위치 정보 표시 → 기본 구조에 반영해두었고, 스케치 주시면 세부 배치를 맞추겠습니다
- 로케이션(구역 위치) 데이터는 현재 DB에 확장 포인트만 잡아두었습니다. 위치 데이터를 어떤 형태로 관리하실지(엑셀, 직접 입력 등) 말씀해주시면 거기에 맞추겠습니다
- 쿠팡 xlsx 새로 다운받아서 지정 폴더에 넣으면 자동으로 데이터가 갱신되도록 구성했습니다
