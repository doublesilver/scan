1) 전달해주신 데이터 분석

2) 서버 API
- 바코드 조회, 상품 검색, 이미지 표시 API를 구성했습니다

- NAS 이미지 연동은 접속 정보 받으면 설정값만 넣으면 바로 연결됩니다

3) PDA 앱
- 화면은 기본 틀만 잡아둔 상태입니다
- 말씀해주신 대로 상품 사진이 크게 나오도록, 그리고 위치 정보도 함께 표시할 수 있도록 구조를 잡았습니다
- 스케치 주시면 거기에 맞춰서 세부 배치 조정하겠습니다
- 스캐너 연동 코드는 작성해두었고 PDA 수령 후 세팅하면 됩니다

4) 말씀해주신 부분 반영
- 발주 담당자 등 불필요 필드는 제외했고, 향후 도매처 연락처/URL 같은 필드를 추가할 수 있도록 확장 구조로 설계했습니다
- 재고 입력이나 위치 표시는 지금 구현하지 않았지만, 이후 추가할 때 기존 구조를 건드리지 않아도 되도록 DB를 설계해두었습니다
- xlsx 파일을 새로 다운받으시면 폴더에 넣기만 하면 자동으로 데이터가 갱신됩니다
