## 1. 코드 품질 검수 및 보안 강화

- 전체 코드를 5가지 관점(물류 도메인, 서버, 앱, 기획, 보안)에서 검수하였습니다
- 이미지 주소를 조작하여 서버 파일에 접근할 수 있는 보안 취약점을 발견하여 차단하였습니다
- 동일 상품에 바코드가 여러 개인 경우 어떤 바코드로 스캔해도 이미지가 정상 표시되도록 수정하였습니다
- 바코드 중복 스캔 방지 처리를 추가하였습니다
- 전수검사 결과 Critical 8건, Warning 30건을 수정하였습니다

## 2. 앱 UI/UX 개선

- 대표님 피드백(이미지 최대, 바코드 끝5자리 볼드, 스페이스쉴드 자동 삭제, 이미지 토글, 한 화면 구성) 반영하였습니다
- Stitch "Tactical Command" 디자인을 적용하여 앱 전체 UI를 개선하였습니다
- 앱 실행 시 서버 연결 상태를 자동 확인하는 시작 화면을 추가하였습니다
- 하단 네비게이션 제거, 에러 메시지 개선, 뒤로가기 처리 등 사용성을 보완하였습니다

## 3. Mini PC 서버 배포 완료

- Mini PC(Windows 11)에 서버를 배포하였습니다
- NSSM으로 Windows 서비스 등록하여 PC 시작 시 서버가 자동으로 실행됩니다
- 서버 상태 모니터링 API를 추가하였습니다

## 4. NAS 실연결 완료

- NAS WebDAV(chominseven.synology.me:58890) 이미지 실연결을 완료하였습니다
- 썸네일(108KB)과 실사 이미지(2.3MB)가 정상적으로 표시됩니다
- NAS xlsx 파일 자동 동기화를 구현하였습니다. 5분 주기로 변경 사항을 자동 반영합니다

## 5. 오프라인 대응

- 창고 내 WiFi 불안정 구간에서도 최근 조회한 상품을 확인할 수 있도록 오프라인 저장(Room DB) 기능을 적용하였습니다
- 서버 연결 실패 시 자동 재시도 기능을 추가하였습니다

## 6. 백업/복구 자동화

- 매일 새벽 3시에 자동으로 데이터를 백업합니다. 최근 7일치를 보관합니다
- 문제 발생 시 명령어 한 줄로 복구할 수 있도록 구성하였습니다

## 7. 재고 수정 API 준비

- 재고 수정용 테이블(stock, stock_log)과 API를 준비하였습니다
- 현재는 사용하지 않으며, 추후 필요하시면 바로 활성화할 수 있습니다

## 8. 운영 가이드 문서

- 서버 시작/중지, 데이터 갱신, PDA 앱 설치, 문제 해결 등을 정리한 운영 가이드 문서를 작성하였습니다
- 납품 시 함께 전달드릴 예정입니다
