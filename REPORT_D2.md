# D+2 클라이언트 보고 멘트

대표님 안녕하세요. 이은석입니다.
착수 이틀째 진행 현황 보고드립니다.

금일은 어제 만들어둔 서버와 앱의 전체 품질을 점검하고, 실제 현장에서 안정적으로 사용할 수 있도록 보완 작업을 진행하였습니다.

1. 앱 품질 검수 및 보안 강화

- 전체 코드를 5가지 관점(물류 도메인, 서버, 앱, 기획, 보안)에서 검수하였습니다
- 이미지 주소를 조작하여 서버 파일에 접근할 수 있는 취약점을 발견하여 차단하였습니다
- 동일 상품에 바코드가 여러 개인 경우 어떤 바코드로 스캔해도 이미지가 정상 표시되도록 수정하였습니다
- 바코드 중복 스캔 방지 처리를 추가하였습니다

2. 앱 시작 화면 추가

- 앱 실행 시 서버 연결 상태를 자동으로 확인하는 시작 화면을 추가하였습니다
- 서버가 꺼져 있거나 WiFi가 끊긴 경우 안내 메시지가 표시됩니다

3. WiFi 불안정 환경 대비

- 창고 내 WiFi가 불안정한 구간에서도 최근 조회한 상품은 계속 확인할 수 있도록 오프라인 저장 기능을 적용하였습니다
- 서버 연결이 끊겼을 때 자동으로 재시도하는 기능을 추가하였습니다

4. 이미지 로딩 속도 개선

- 이미지를 PDA 화면 크기에 맞게 줄여서 전송하도록 변경하여 로딩 속도를 개선하였습니다

5. 운영 가이드 문서 작성

- 서버 시작/중지, 데이터 갱신, PDA 앱 설치, 문제 해결 등을 정리한 운영 가이드 문서를 작성하였습니다
- 납품 시 함께 전달드릴 예정입니다

6. 자동 백업/복구

- 매일 자동으로 데이터를 백업하고, 문제 발생 시 명령어 한 줄로 복구할 수 있도록 구성하였습니다
- 최근 7일치 백업을 보관합니다

현재 진행 상황

- 전체 8개 마일스톤 중 7개 완료, 85% 진행
- 남은 작업: PDA 실기기 테스트, NAS 실연결 확인, Mini PC 서버 설치

확인 요청 드리는 사항

1. NAS WebDAV 접속 주소 및 계정 정보 -- 이미지 실연결에 필요합니다
2. PDA(TC60) 택배 발송 일정 -- 앱 설치 및 실기기 스캔 테스트 진행을 위해 필요합니다
