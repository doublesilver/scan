확인 요청 드리는 사항

아래 내용을 확인해주시면 나머지 작업을 빠르게 마무리할 수 있습니다.

1. 실사 이미지 관리 방식

- NAS 확인 결과, img/ 폴더에 바코드\_real.jpg 형태의 실사 사진이 10장 정도 있고, real_image/ 폴더에도 별도로 2장이 있습니다
- 앞으로 실사 사진을 올리실 때 어느 폴더를 기준으로 하시면 될까요? 어디에 올리시든 앱에서 자동 인식되게 처리해드리겠습니다

2. 발주하기 URL 데이터

- 말씀하신 상품별 구매 URL이 매칭된 데이터를 공유해주시면 발주 버튼을 바로 연동하겠습니다
- 엑셀 형태이시면 어떤 컬럼에 URL이 들어가 있는지 알려주시면 됩니다

3. 신규 상품 이미지 확보 방법

- 현재 img/ 폴더에 11,689장의 이미지가 있는데, 신규 SKU가 추가될 때 이미지는 어떻게 확보하고 계신가요?
- 서플라이허브 접근 방법을 공유해주시면 이미지 자동 수집 방법을 확인해보겠습니다

4. 재고 수량 입력 (참고)

- 말씀하신 재고 입력 기능을 준비해두고 있는데, 주로 어떤 상황에서 수량을 입력하시는지 간단히 알려주시면 화면 구성에 반영하겠습니다
- 예: 새 물건 입고 시 / 정기 재고 실사 시 / 출고 시 등

편하신 시점에 확인 부탁드립니다. 감사합니다.
