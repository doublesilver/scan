필드와 관계입니다.
테이블 상세
테이블 주요 필드 관계
제품 마스터 ID, 제품명, 브랜드, 제품군, 상태 1:N → SKU
속성 재질, 커넥터, 체결방식, 폭, 길이 N:1 → 제품
SKU ID, 컬러, 사이즈, 바코드, 재고 N:1 → 제품
호환 모델 모델명, 워치사이즈 M:N ↔ SKU
가격 공급가, 도매가, 판매가, 채널별 N:1 → SKU
콘텐츠 자동 상품명, 옵션명, 키워드 N:1 → SKU
M:N 관계로 기종 추가 시 컬럼이 아닌 행만 추가
기종이 늘어나도 시트 구조 변경 없이 호환 모델 테이블에 행만 추가합니다.
