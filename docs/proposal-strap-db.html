테이블 주요 필드 관계
제품 마스터 ID, 제품명, 브랜드, 제품군, 상태 1:N → SKU, 속성
속성 재질, 커넥터, 체결방식, 폭, 길이 N:1 → 제품
SKU ID, 컬러, 사이즈, 바코드, 재고 N:1 → 제품
호환 모델 모델명, 워치사이즈 M:N ↔ SKU
가격 공급가, 도매가, 판매가, 채널별 N:1 → SKU
콘텐츠 자동 상품명, 옵션명, 키워드 N:1 → SKU
M:N 관계를 통해 기종 추가 시 컬럼이 아닌 행만 추가하면 됩니다.
04
핵심 기능

SKU 조회 시스템

  • 바코드/제품명 입력 → 제품정보+이미지+속성 즉시 출력
  • 기존 PDA 스캐너 시스템과 연동 가능

다차원 필터링

  • 재질별, 커넥터별, 호환 모델별, 가격대별
  • 복합 조건 지원 (예: 실리콘 + 갤럭시워치 + 20mm)

상품명/옵션명 자동 생성

  • 규칙: "스페이스쉴드 + 재질 + 체결 + 모델 + 특징"
  • 네이버/쿠팡 글자수 제한 자동 대응
  • 내부코드 병행 관리

구글시트 양방향 동기화

  • DB ↔ 시트 자동 반영
  • 충돌 시 DB 우선 + 알림
