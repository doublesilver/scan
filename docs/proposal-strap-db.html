현황 이해

04
핵심 기능

SKU 조회

  • 제품명 또는 내부코드로 검색하면 해당 SKU의 모든 정보가 한 화면에 표시됩니다

다차원 필터링

  • 예를 들어 '실리콘 재질이면서 갤럭시워치에 맞는 20mm 스트랩'을 한번에 찾을 수 있습니다
  • 재질별, 커넥터별, 호환 모델별, 가격대별 복합 조건 지원

상품명/옵션명 자동 생성 및 콘텐츠 활용

  • 규칙: "스페이스쉴드 + 재질 + 체결 + 모델 + 특징" 조합 자동화
  • 네이버/쿠팡 글자수 제한 자동 대응
  • 내부코드 병행 관리
  • 생성된 상품명·옵션명은 상세페이지 콘텐츠 제작에 직접 활용 가능

구글시트 연동 구조

  • 구글시트 → DB 자동 동기화 (기존 입력 방식 유지)
  • 웹 UI에서도 직접 입력 가능 — 두 방식 병행 지원
  • 충돌 시 DB 우선 + 알림
