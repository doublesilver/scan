왜 스캔이 필요한가

11,821개 SKU를 사람이 육안으로 식별하려면 2~3개월의 숙련 기간이 필요합니다. 바코드 스캔은 이 숙련 과정을 즉시 해결합니다. 신입 직원도 첫날부터 정확한 상품 식별이 가능합니다.

PDA는 NAS에 직접 접근하지 않습니다
모든 요청은 Mini PC 서버를 경유합니다. NAS 접근 권한 관리가 단순해지고 네트워크 부하를 서버에서 제어할 수 있습니다.

모듈화 설계: 각 기능(조회, 파싱, 이미지 처리)이 독립된 모듈로 구성됩니다. 이후 기능을 추가하거나 기존 사내 프로그램과 연동할 때 기존 구조를 변경하지 않고 모듈 단위로 붙일 수 있는 구조입니다.

헤더 불일치 문제
쿠팡 서플라이어 허브에서 다운로드하는 파일은 시점에 따라 컬럼 이름이나 순서가 달라질 수 있습니다. 이를 대비한 두 가지 방안을 제시합니다.
방안 A: 자동 매칭 (권장)
컬럼명 유사도 분석으로 자동 매핑
매칭 실패 시 관리자에게 알림
초기 설정 후 추가 작업 불필요
방안 B: 수동 지정
관리자가 컬럼 매핑 규칙을 직접 지정
파일 형식이 바뀌면 재설정 필요
정확도 100% 보장
대표님 판단에 따라 선택 가능합니다
두 방안 모두 구현 가능하며, 운영 편의성과 정확도 중 우선순위에 따라 결정하시면 됩니다. 혼합 적용도 가능합니다.
