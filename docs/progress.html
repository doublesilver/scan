테이블 건수 역할
PRODUCT 11,821건 상품 마스터 (SKU ID, 상품명, extra JSON)
BARCODE 11,821건 바코드 매핑 (타입: EAN-13/QR/CODE128)
IMAGE 11,673건 이미지 경로 (바코드 기준, 썸네일/실사)
PRODUCT_FTS — 전문 검색 인덱스 (10만건 이상 대응)
확장 설계: DB 스키마 마이그레이션 체계(db_version)를 도입하여 향후 테이블/컬럼 추가 시 자동 적용됩니다. 읽기/쓰기 커넥션 분리(WAL 모드), FTS5 전문 검색, 배치 파싱(500건 단위)으로 10만건 이상 데이터에서도 안정적으로 동작합니다. 재고 입력(INVENTORY), 위치 표시(LOCATION) 등은 테이블 추가만으로 확장 가능합니다.

적재 데이터

쿠팡 SKU 파일 (17개 컬럼 중 주요 항목)

컬럼 활용 예시
SKU ID 상품 고유 키 1863995
상품명 상품 식별 쵸미세븐 무당벌레 청소기 레드
바코드 스캔 매칭 8809461170008
최소구매수량 참고 정보 1
중량 (g) 참고 정보 160

발주 담당자 등 불필요 필드는 제외하였으며, 필요하신 필드가 있으시면 추가 가능합니다.

헤더 자동 인식: 기존 프로그램에서 발생하던 "SKU ID/상품명/바코드 열을 찾지 못했습니다" 오류는 쿠팡에서 다운로드 시점에 따라 컬럼명이 달라지는 것이 원인이었습니다. 컬럼명 유사도 분석으로 자동 매칭하도록 대응하였습니다.
