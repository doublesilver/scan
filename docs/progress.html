04
프로젝트 요약

이 시스템이 뭔가요?

PDA(휴대용 스캐너)로 상품 바코드를 찍으면, 상품 이름/이미지가 바로 화면에 뜨는 시스템입니다. 기존 PC 프로그램(scan13)은 그대로 유지하고, PDA 앱을 추가로 설치합니다.

기존과 뭐가 달라지나요?

Before (현재)
  • PC 앞에 가서 조회해야 함
  • USB 스캐너를 PC에 연결해서 사용
  • 이미지가 작게 표시됨
  • 엑셀 헤더가 바뀌면 인식 실패
After (도입 후)
  • PDA 들고 다니며 현장에서 바로 조회
  • PDA 버튼만 누르면 스캔
  • 이미지가 화면 전체에 크게 표시
  • 엑셀 양식 바뀌어도 자동 인식

사용 흐름

실제 현장에서 PDA를 사용하는 순서입니다.

01
PDA 전원 켜기
전원 켜면 앱이 자동 실행됩니다. 별도 로그인 없음.
02
바코드 스캔
PDA 옆면 버튼을 누르면 스캐너가 작동합니다.
03
상품 정보 확인
상품명, 바코드, 이미지가 화면에 표시됩니다.
0.3초 이내
04
이미지 확인
이미지를 터치하면 쿠팡 사진 / 실사 사진이 전환됩니다.
05
다음 상품 스캔
바로 다음 상품 바코드를 스캔하면 됩니다. 뒤로가기 필요 없음.
06
예상 시나리오

실제 업무에서 이런 상황이 생기면 이렇게 됩니다.

1
신입 직원이 상품을 찾아야 할 때
상품명만 보고는 어떤 물건인지 모르겠다
➔
바코드 스캔하면 상품 사진이 크게 표시되어 바로 확인 가능
2
같은 상품인데 바코드가 다를 때
하나의 상품에 바코드가 3~5개 붙어 있다
➔
어떤 바코드로 스캔해도 같은 상품 정보가 표시됨
3
WiFi가 끊기는 구간에서
5층 건물 이동 중 WiFi 신호가 약해진다
➔
이전에 조회한 상품은 인터넷 없이도 확인 가능 (자동 저장)
4
새 상품이 입고되었을 때
쿠팡에서 새 엑셀 파일을 다운받았다
➔
NAS에 엑셀 파일 올리면 5분 내 자동 반영. 별도 작업 없음
5
서버가 꺼졌을 때
Mini PC가 재부팅되거나 서버가 멈췄다
➔
앱에 안내 메시지 표시 + 이전 조회 상품은 캐시로 계속 사용. 서버는 PC 시작 시 자동 실행됨
07
주요 기능
바코드 스캔
PDA 옆면 버튼을 누르면 0.3초 안에 상품 정보가 표시됩니다. EAN-13 바코드 기본 지원, QR 코드도 대응 가능합니다.
상품 검색
바코드가 없는 경우 상품명으로 직접 검색할 수 있습니다. 현재 11,821건 등록, 10만건까지 대응 가능합니다.
이미지 확인
쿠팡 썸네일과 실사 사진을 화면 가득 크게 표시합니다. 이미지를 터치하면 썸네일/실사가 전환됩니다.
자동 데이터 갱신
NAS에 새 엑셀 파일을 올리면 5분 이내 자동 반영됩니다. 별도 작업이 필요 없습니다.
오프라인 지원
WiFi가 불안정해도 최근에 조회한 상품은 PDA에 자동 저장되어 계속 확인할 수 있습니다.
자동 백업
매일 새벽 3시에 전체 데이터를 자동 백업합니다. 최근 7일치를 보관하며, 문제 발생 시 즉시 복구 가능합니다.
08
데이터 관리

기존에 하시던 방식 그대로입니다. 추가 작업이 필요 없습니다.

상품 데이터는 어떻게 관리하나요?

  1. 쿠팡 서플라이어허브에서 엑셀(xlsx) 다운로드
  2. NAS의 기존 폴더에 저장
  3. 끝 -- 서버가 자동으로 감지해서 반영합니다

새 상품 추가, 기존 상품 정보 변경 모두 자동 처리됩니다. 쿠팡에서 엑셀 양식이 바뀌어도 컬럼을 자동 인식합니다.

이미지는?

  • 쿠팡 썸네일
    NAS의 img/ 폴더에 넣으면 자동 연결
  • 실사 사진
    NAS의 real_image/ 폴더에 넣으면 자동 연결

이미지 파일명과 바코드의 매핑은 codepath.xlsx가 관리합니다. 기존 방식과 동일합니다.

자주 묻는 질문
인터넷이 필요한가요?
외부 인터넷은 필요 없습니다. 사무실 내부 WiFi만 있으면 됩니다. PDA와 Mini PC가 같은 WiFi에 연결되어 있으면 동작합니다.
기존 프로그램(scan13)은 어떻게 되나요?
그대로 유지됩니다. PDA 앱은 별도로 추가되는 것이고, 기존 Mini PC의 USB 스캐너와 scan13 프로그램은 계속 사용할 수 있습니다.
상품이 추가되면 어떻게 하나요?
기존과 동일합니다. 쿠팡에서 엑셀 다운로드 후 NAS에 저장하면 끝. 서버가 5분마다 자동으로 확인하여 반영합니다.
PDA 여러 대로 동시에 사용 가능한가요?
같은 WiFi에 연결된 PDA라면 대수 제한 없이 동시 사용 가능합니다.
데이터가 날아가면요?
매일 새벽 3시에 자동 백업됩니다. 최근 7일치를 보관하며, 문제 발생 시 명령어 한 줄로 복구할 수 있습니다.
