1. 서버 연결

PDA가 사내 WiFi에 연결돼 있어야 서버와 통신합니다.

1.1 자동 감지 (권장)

  1. 메인 화면 우측 상단 톱니바퀴(⚙) 탭 → 설정 화면
  2. 서버 자동 감지 탭
  3. 자동으로 같은 네트워크의 서버를 찾아서 주소 입력됨
  4. 연결 테스트 탭 → "연결 성공" 확인 후 뒤로

1.2 수동 입력

자동 감지가 실패하면 서버 주소를 직접 입력합니다.

  1. 설정 → 서버 URL 입력칸 탭
  2. 받은 주소 입력 (예: http://...:8000)
  3. 저장 → 연결 테스트 → "연결 성공"

1.3 연결 실패 시

  • WiFi 연결 상태 확인 (사내 WiFi인지)
  • 주소에 오타 없는지 (http://, :8000 빠짐 주의)
  • 그래도 실패 → 12장 응급 복구
💡 외부망(개인 핫스팟, LTE)에서는 서버 접속 불가. 사내 WiFi 필요.

2. 바코드 스캔

2.1 절차

  1. 바코드를 PDA 정면 위쪽 레이저 창을 향하게
  2. PDA 옆면/윗면의 노란색 트리거 버튼 누름
  3. 빨간 빛 + "삑" 소리 → 0.5초 내 메인 하단에 결과 카드 표시

2.2 결과 카드

  • 상품 이미지
  • 상품명 / 바코드 / SKU
  • 인쇄 버튼 (1장 즉시 인쇄)
  • 카드 자체 탭 → 상품 상세 페이지

2.3 스캔이 안 될 때

  • 트리거 버튼 끝까지 눌렀는지 (살짝 누르면 안 됨)
  • 빨간 빛이 안 나오면 → PDA 재부팅 (전원 길게 → 다시 시작)
  • 거리 10~20cm 유지 (너무 가까우면 인식 안 됨)
  • 바코드 구겨짐/오염 확인 → 닦기
  • 모두 안 되면 → 검색창에 바코드 번호 직접 입력

3. 텍스트 검색

3.1 입력 가능한 것

  • 상품명 일부: "케이스", "충전기" 같은 단어
  • 바코드 번호: 8~13자리 숫자
  • SKU 번호: 쿠팡 SKU 번호
  • 박스 번호: BOX-XXXX 형식

3.2 절차

  1. 메인 검색창 탭 → 키보드 올라옴
  2. 검색어 입력 → 검색 버튼 또는 키보드의 검색 키
  3. 결과 카드 목록에서 원하는 상품 탭 → 상세 페이지

3.3 결과가 너무 많으면

  • 더 구체적인 단어 사용 (예: "케이스" → "아이폰 15 케이스")
  • 바코드/SKU로 검색하면 정확히 1건만 나옴

4. 상품 상세 페이지

4.1 화면 구성

  • 상단: 상품명, SKU, 카테고리, 바코드
  • 중앙: 상품 이미지 (탭하면 썸네일 ↔ 실사 사진 전환)
  • 하단 네비바 3버튼: 도면 / 발주 / 인쇄

4.2 바코드 복사

  • 상단 바코드 텍스트를 약 1초간 길게 누름
  • "바코드가 복사되었습니다" 토스트 → 다른 앱에 붙여넣기 가능

4.3 도면으로 바로가기

하단 도면 버튼 → 창고 도면이 팝업으로 뜨고 이 상품 위치가 노란색으로 강조됨

5. 발주 (구글시트 자동 등록)

5.1 절차

  1. 상품 상세에서 발주 버튼 탭
  2. 수량 입력 다이얼로그가 뜸
    빠른 선택: 1 / 10 / 50 / 100 또는 직접 입력 (1~999)
  3. 발주 버튼 탭
  4. "발주 요청 중..." → "발주 추가 완료" 토스트 (1~2초)

5.2 발주가 어디로 가나?

  • 구글시트에 자동으로 한 줄 추가됨
  • 추가 정보: 날짜시간 / SKU / 상품명 / 수량
  • 시트는 PC 브라우저로 열어서 확인
  • ⚠️ PDA 앱에는 발주 내역 조회 기능이 없습니다.

5.3 발주 실패 메시지별 대응

메시지 대응
발주 실패: 시트 연동 실패 서버↔구글 인증 문제, 잠시 후 다시 시도
발주 실패 (HTTP 500) 1분 후 다시 시도
발주 요청 실패: 연결 오류 WiFi 확인 후 다시 시도
토스트가 아예 안 뜸 앱 강제종료 → 재실행

6. 라벨 인쇄

라벨 인쇄는 두 가지 경로가 있습니다.

6.1 경로 A — 메인 스캔 결과 카드에서 즉시 인쇄

  1. 바코드 스캔
  2. 메인 하단 결과 카드의 인쇄 버튼 탭
  3. 즉시 1장 인쇄 (수량 입력 없음)

6.2 경로 B — 상세 페이지에서 수량 지정 인쇄

  1. 상품 상세에서 인쇄 탭 (하단 네비바 우측)
  2. 수량 입력 (1 / 5 / 10 / 100 빠른 선택 또는 직접 입력 1~100)
  3. 인쇄 탭

6.3 인쇄 흐름

  • PDA → 미니PC 서버 → 물류PC의 인쇄 에이전트 → TSC 라벨 프린터
  • 정상 시 1~2초 내 라벨 출력 + "N장 인쇄 완료" 토스트

6.4 인쇄 실패 메시지별 대응

메시지 원인 대응
물류PC 응답 없음 (30초 타임아웃) 물류PC의 print_agent.py 가 꺼짐 아래 6.5 절차
프린터 오프라인 — 전원/케이블 확인 프린터 자체 문제 전원/USB 확인 후 재시도
상품명과 라벨이 다름 템플릿 매칭 오류 앱 재시작 후 재시도
라벨 템플릿 없음 신규 상품, 템플릿 미등록 템플릿 등록 필요 (현장 조치 불가)

6.5 "물류PC 응답 없음" 직접 살리기 (3분)

물류PC에 가서 다음 절차를 따라하세요.

  1. 물류PC 모니터 앞에 앉아서 마우스 흔들어 화면 깨우기
  2. 키보드 Win 키 → cmd 입력 → 엔터 (검은 창 = 명령 프롬프트가 뜸)
  3. 검은 창에 아래 한 줄을 그대로 입력하고 엔터: 
    pythonw C:\print_agent.py
  4. 별다른 메시지 없이 명령이 끝남 (정상)
  5. 검은 창은 닫지 말고 최소화 (창 닫으면 같이 꺼짐)
  6. PDA로 돌아와서 라벨 인쇄 다시 시도
pythonw 인식되지 않습니다 오류 시
아래 두 가지 중 하나로 시도: 
C:\Python311\pythonw.exe C:\print_agent.py
python C:\print_agent.py
그래도 안 되면 물류PC를 재부팅 후 다시 시도.

PC 재부팅 후 자동 실행 등록 (선택)

명령 프롬프트에 한 번만 실행:

reg add "HKCU\Software\Microsoft\Windows\CurrentVersion\Run" /v PrintAgent /t REG_SZ /d "pythonw.exe C:\print_agent.py" /f

7. 창고 도면

7.1 도면 여는 방법

  • 메인 화면 중앙: 항상 도면이 보임
  • 상품 상세 → 도면 버튼: 그 상품 위치를 노란색으로 강조해서 팝업

7.2 색상 의미

색상 의미
🟡 노란색 (깜빡임) 현재 검색 중인 상품 위치
🟢 녹색 상품이 보관된 셀
⬜ 회색 비어있는 셀
🔲 검정 통로
🟦 파란색 테이블/작업대
🟥 빨간색 물류PC 위치

7.3 도면 조작

  • 두 손가락 벌리기: 확대 (핀치줌)
  • 두 손가락 좁히기: 축소
  • 셀 탭: 그 셀의 상세 페이지로 이동

8. 셀 상세 / 사진 변경

8.1 셀 상세 화면

도면에서 셀을 탭하면 진입.

  • 풀스크린 셀 사진 (없으면 "사진 없음 / 탭하여 촬영" 표시)
  • 화면 하단에 셀 위치 라벨 (예: "A구역 1-1")
  • 사진을 두 손가락으로 확대 가능 (핀치줌)
  • 사진 더블탭 → 1배 크기로 복귀

8.2 하단 네비바 3버튼

버튼 동작
도면 도면 팝업 (다른 셀로 이동 가능)
가운데 (셀 라벨) 표시만 — 클릭 동작 없음
편집 셀 사진 변경 흐름 시작

8.3 셀 사진 새로 찍기 / 변경

  1. 셀 상세에서 편집 탭
  2. "새 사진을 업로드하시겠습니까?" → 확인 탭
  3. 카메라 또는 갤러리 선택
  4. 촬영 또는 사진 선택 → 자동 업로드
  5. "업로드 완료" 토스트 → 사진이 즉시 갱신됨

8.4 사진 촬영 팁

  • 셀 안 상품들이 한 화면에 다 보이도록 약간 떨어져서 촬영
  • 어두우면 PDA 카메라 화면의 플래시 켜기
  • 셀 라벨도 같이 보이게 찍으면 다음에 찾기 쉬움

8.5 사진 업로드 실패 시

증상 대응
"업로드 실패" 토스트 편집 → 사진 변경 다시 시도
계속 실패 WiFi 신호 확인 (PDA 우측 위 막대)
사진이 90도 회전됨 알려진 이슈, 갤러리 앱에서 회전시켜 저장 후 갤러리로 업로드
모두 실패 PDA 재부팅 후 다시 시도

9. 박스(QR) 스캔

9.1 스캔 절차

  1. 외박스의 QR 코드를 PDA 정면을 향하게
  2. 트리거 버튼 → "삑" 소리
  3. 박스 상세 페이지로 자동 이동

또는 검색창에 박스 번호 직접 입력 (예: BOX-1234)

9.2 박스 상세 화면

  • 상단: 박스 대표 상품명 / 구역·위치 태그
  • 중앙: 박스 사진 + 도면 (박스 위치 노란색 강조)
  • 옵션 이미지: 한국 옵션 이미지
  • 발주 이미지: 중국 측 발주용 옵션 이미지
  • 외부 링크 4버튼:
    • N → 네이버 (스마트스토어)
    • 🚀 → 쿠팡
    • 1688 → 중국 1688 구매처
    • FLOW → FLOW 링크
    • 해당 링크가 없는 항목은 비활성됨

9.3 박스 상세 하단 네비바

  • 도면: 박스 위치를 도면에서 보기
  • 구매하기: 쿠팡 상품 페이지를 외부 브라우저로 열기 (발주 시트와는 무관)
  • 인쇄: 박스 라벨 인쇄

10. 상품 배치 (셀에 새로 등록 / 위치 변경)

메인 화면 하단 🔄 배치 버튼 → 배치 화면 진입. 상품을 새로 셀에 등록하거나 기존 위치를 다른 셀로 옮길 때 사용.

10.1 절차

  1. 메인 하단 🔄 배치 탭
  2. 배치 화면에서 바코드 스캔 (또는 검색창에 직접 입력)
    • 일반 바코드 → 상품 배치 모드
    • BOX-XXXX QR → 외박스 배치 모드
  3. 상단에 상품/박스 정보 카드 표시 + 현재 위치 표시
  4. 셀 선택 버튼 탭 → 도면 팝업
  5. 도면에서 원하는 셀 탭 → 선택된 셀 정보 표시
  6. (선택) 셀 사진 / 박스 사진 미리 등록 — 촬영 또는 갤러리 버튼
  7. 배치 확정 탭 → 서버에 저장 → "배치 완료" 토스트

10.2 다시 시작하려면

  • 잘못 스캔했거나 다른 상품으로 바꾸려면 다시 시작 버튼 탭 → 처음 스캔 단계로

10.3 배치 실패 시

  • WiFi 신호 확인 후 다시 시도
  • "셀 선택" 후 도면이 안 뜨면 앱 강제종료 → 재실행
  • 같은 셀에 너무 많은 상품을 배치하지 않도록 주의 (도면 색상이 녹색 → 가득참 표시)

10.4 외박스(BOX-) 배치 팁

  • 일반 상품 배치보다 박스 배치가 더 자주 사용됨
  • 외박스 QR을 스캔하고 셀을 선택하면, 그 박스에 담긴 모든 상품의 위치가 한꺼번에 갱신됨

11. 앱 업데이트

11.1 자동 업데이트

  • 앱 실행 시 새 버전이 있으면 다이얼로그 자동 표시
  • 업데이트 탭 → 자동 다운로드 (10~30초) → 설치 탭 → 앱 자동 재시작

11.2 알림이 안 뜰 때

  1. 최근 앱 (PDA 하단 ▢ 버튼) → 앱을 위로 쓸어서 종료
  2. 앱 다시 실행
  3. 안 뜨면 → 이미 최신이거나 서버 연결 안 됨

11.3 현재 버전 확인

설정 화면 (메인 우측 상단 톱니바퀴) 하단에 표시됨. 예: v5.4.7

12. 자주 묻는 질문 (FAQ)

Q1. 바코드 스캔이 안 돼요
트리거 버튼 끝까지 → 빨간 빛 확인 → 거리 10~20cm → 바코드 청소 → 검색창에 직접 입력
Q2. "서버 연결 실패" 떠요
사내 WiFi 연결 확인 → 설정 → 자동 감지 또는 주소 재확인 → 연결 테스트 → 그래도 실패 시 미니PC 재부팅 검토
Q3. 상품을 찾을 수 없다고 떠요
다른 검색어 (상품명/SKU)로 다시 시도 → 신규 상품은 서버 DB 등록 필요 (현장 조치 불가)
Q4. 라벨이 안 찍혀요 — "물류PC 응답 없음"
6.5 절차 그대로 (cmd 열고 pythonw C:\print_agent.py)
Q5. 발주 시트에 안 들어가요
"발주 추가 완료" 토스트 떴는지 확인 → 시트 새로고침 → 1~2초 지연 가능 → "시트 연동 실패" 시 잠시 후 재시도
Q6. 셀 사진이 옛날 거로 보여요
다른 PDA에서 변경된 경우 최대 1분 지연 → 뒤로가기 → 다시 진입하면 즉시 갱신
Q7. 사진이 90도 돌아서 업로드돼요
알려진 이슈, 다음 업데이트에서 수정 예정. 임시로 갤러리 앱에서 회전시켜 저장 후 갤러리로 업로드
Q8. 앱이 멈추거나 느려요
최근 앱 (▢) 위로 쓸기 → 재실행 → PDA 재부팅 → 새 업데이트 확인
Q9. PDA 화면이 너무 빨리 꺼져요
PDA 시스템 설정 → 디스플레이 → 화면 자동 꺼짐 시간 늘리기 (앱에서 제어 안 함)
Q10. 다른 사람도 같은 PDA 써도 되나요?
네, 누구나 사용 가능. 단 발주/사진 변경한 사람이 누구인지는 기록 안 됨
Q11. WiFi가 자꾸 끊겨요
신호가 약한 곳일 수 있음 (구역 끝, 컨테이너 안 등) → 신호 좋은 곳으로 이동
Q12. 도면이 안 보이거나 깨져 보여요
뒤로가기 → 다시 진입 → 그래도 안 되면 앱 강제종료 → 재실행 → 도면 자체 문제면 앱 재시작

13. ⚡ 응급 복구

13.1 PDA 한 대만 안 됨

  1. 앱 강제종료 → 재실행 (▢ → 위로 쓸기)
  2. WiFi 끄기 → 5초 → 다시 켜기
  3. PDA 재부팅 (전원 길게 → 다시 시작)
  4. 다른 PDA로 동일 작업 시도 → 다른 PDA가 정상이면 이 PDA만의 문제

13.2 모든 PDA가 안 됨 (서버 다운 가능성)

  1. 다른 직원 PDA도 확인 — 모두 동일 증상이면 서버 문제
  2. 미니PC 모니터 확인 — 빨간 에러 메시지? 인터넷 케이블 LED 깜빡임?
  3. 미니PC 재부팅 → 시작 → 전원 → 다시 시작 → 2~3분 후 PDA에서 시도

13.3 라벨만 안 찍힘 (스캔/검색은 정상)

  1. 물류PC가 켜져 있는지 확인
  2. 프린터 전원/USB 확인
  3. 6.5 절차로 인쇄 에이전트 다시 실행
  4. PDA에서 라벨 다시 인쇄
  5. 그래도 안 되면 라벨 템플릿(*.btw) 누락 가능 (현장 조치 불가)

13.4 발주만 안 됨

  1. WiFi 신호 확인
  2. 1분 후 다시 시도 (구글시트 일시 지연 가능)
  3. PC 브라우저로 시트 직접 열어보기 — 시트 자체가 안 열리면 구글 측 문제
  4. 그래도 안 되면 미니PC 재부팅 검토

13.5 도면이 이상하게 보임

  1. 앱 강제종료 → 재실행
  2. 톱니바퀴 → 메인 다시 진입
  3. 그래도 이상하면 도면 데이터 손상 가능 (현장 조치 불가)
