# 물류창고 PDA 스캐너 앱 사용 가이드

대상: 창고 현장 직원 (IT 비전공)
버전: v5.4.7 기준

---

## 1. 첫 화면 둘러보기

PDA 전원을 켜면 앱이 자동 실행됩니다. 화면 구성을 먼저 익혀두세요.

### 메인 화면

```
┌─────────────────────────────────────────┐
│  [검색창]                      [🔍] [⚙]  │ ← 우측 위 톱니바퀴 = 설정
├─────────────────────────────────────────┤
│                                         │
│         📐 창고 도면 (격자)              │ ← 중앙: 도면
│         색상별 셀 표시                  │
│                                         │
├─────────────────────────────────────────┤
│   [🔄 배치]   [🗂 박스]                 │ ← 하단 네비
└─────────────────────────────────────────┘
```

- **검색창**: 바코드/상품명 입력
- **톱니바퀴(⚙)**: 서버 주소·연결 설정
- **중앙 도면**: 창고 셀(선반) 위치 시각화
- **배치**: 상품을 셀에 배치하는 화면
- **박스**: 박스 등록·관리 화면

## 2. 서버 주소 설정 (네트워크 변경 시 필수)

### 2.1 먼저 PDA가 어떤 WiFi에 연결됐는지 확인

1. PDA 홈 화면 위쪽 상태바를 **아래로 두 번 쓸어내림** (퀵 설정 펼침)
2. **WiFi 아이콘 길게 누름** → WiFi 설정 화면
3. 화면 맨 위 **"연결됨"** 옆에 표시된 **WiFi 이름**을 확인
   - 예: `WAREHOUSE_5G`, `Office_WiFi`, `iPhone_핫스팟` 등

### 2.2 WiFi에 따라 입력할 서버 주소

| 연결된 WiFi                         | 입력할 서버 주소                           |
| ----------------------------------- | ------------------------------------------ |
| 창고 내부 WiFi (예: `WAREHOUSE_5G`) | `http://100.125.17.60:8000`                |
| 사무실 WiFi                         | `http://100.125.17.60:8000`                |
| 외부망 / 핫스팟                     | **서버 접속 불가** — 창고 WiFi로 변경 필요 |

> 💡 미니PC 서버는 **Tailscale**이라는 보안 네트워크로 운영됩니다. 외부에서는 접속이 안 되고, 회사 WiFi에 붙어 있어야 합니다.

### 2.3 서버 주소 입력 절차

1. 메인 화면 우측 상단 **톱니바퀴(⚙)** 탭
2. **서버 URL** 입력칸을 탭하고 위 표의 주소 그대로 입력
3. **저장** 탭
4. **연결 테스트** 탭 → "연결 성공" 토스트가 뜨면 OK
5. 뒤로가기로 메인 화면 복귀

### 2.4 "연결 실패" 토스트가 뜨면

- 입력한 주소에 오타 없는지 다시 확인 (`http://`, `:8000`)
- WiFi 다시 한 번 확인 (위 2.1)
- 그래도 실패 → **12장 응급 복구** 참고

## 3. 바코드 스캔으로 상품 찾기

### 3.1 스캔 절차

1. 상품의 바코드(검은색 줄무늬)를 PDA 정면 위쪽 **레이저 창**을 향하게
2. PDA 양옆 또는 윗면의 **노란색 트리거 버튼** 누름
3. 빨간 빛이 나오면서 "삑" 소리 → 0.5초 내 상품 정보 표시

### 3.2 스캔 결과 화면

스캔 성공 시 메인 화면 하단에 카드가 뜹니다:

- 상품명 / 카테고리 / 위치 (예: "A구역 1-2")
- 상품 이미지
- **(카드를 탭하면 상세 페이지로 이동)**

### 3.3 스캔이 안 될 때

| 증상                          | 해결                                                                               |
| ----------------------------- | ---------------------------------------------------------------------------------- |
| 빨간 빛이 안 나옴             | 트리거 버튼을 끝까지 눌렀는지 확인 (살짝 누르면 안 됨)                             |
| 빨간 빛은 나오는데 인식 안 됨 | 바코드와 PDA 거리 10~20cm 유지, 너무 가까우면 안 됨                                |
| 계속 실패                     | 바코드 구겨짐/오염 → 수건으로 닦기 → 그래도 안 되면 검색창에 바코드 번호 직접 입력 |
| 모든 게 안 됨                 | PDA 재부팅 (전원 길게 → "다시 시작")                                               |

## 4. 텍스트 검색 (스캔 못 할 때)

### 4.1 검색창에 입력 가능한 것

- **상품명 일부**: "케이스", "충전기" 같은 단어
- **바코드 번호**: 8~13자리 숫자 (스캔 안 될 때 직접 입력)
- **SKU 번호**: 쿠팡 SKU 번호

### 4.2 검색 절차

1. 메인 화면 검색창 탭 → 키보드 올라옴
2. 검색어 입력 → 키보드의 **🔍 돋보기** 버튼 탭
3. 결과 카드 목록이 나옴
4. 원하는 상품 카드 탭 → 상세 페이지

### 4.3 결과가 너무 많을 때

- 상품명을 더 구체적으로 입력 (예: "케이스" → "아이폰 15 케이스")
- 바코드/SKU로 검색하면 정확히 1건만 나옴

## 5. 상품 상세 페이지

### 5.1 페이지 구성

```
┌─────────────────────────────────────────┐
│  [←]  상품명                            │ ← 좌측 위 ← = 뒤로
│       SKU / 카테고리 / 바코드           │
├─────────────────────────────────────────┤
│                                         │
│            상품 이미지                  │ ← 탭 = 썸네일↔실사 전환
│                                         │
├─────────────────────────────────────────┤
│   [📐 도면]  [🛒 발주]  [🏷 라벨]        │ ← 하단 네비 3버튼
└─────────────────────────────────────────┘
```

### 5.2 영역별 기능

| 영역              | 동작                                                             |
| ----------------- | ---------------------------------------------------------------- |
| 상단 상품명       | 정보 표시만                                                      |
| **바코드 텍스트** | **길게 누르면** 클립보드 복사 ("바코드가 복사되었습니다" 토스트) |
| 중앙 이미지       | 탭하면 썸네일 ↔ 실사 사진 전환                                   |
| **도면** 버튼     | 창고 도면을 띄워서 이 상품 위치(노란색)로 이동                   |
| **발주** 버튼     | 구글시트에 발주 정보 자동 추가 → **6장**                         |
| **라벨** 버튼     | 바코드 라벨 인쇄 → **7장**                                       |

## 6. 발주 (구글시트 자동 등록)

### 6.1 발주 절차

1. 상품 상세 페이지에서 **🛒 발주** 버튼 탭
2. **수량 입력 다이얼로그**가 뜸:
   ```
   ┌─────────────────────────┐
   │   발주 요청              │
   │   수량을 선택하거나      │
   │   직접 입력하세요        │
   │                         │
   │  [1] [10] [50] [100]    │ ← 빠른 선택
   │  [        1        ]    │ ← 직접 입력 (1~999)
   │                         │
   │       [취소] [발주]      │
   └─────────────────────────┘
   ```
3. **빠른 선택** 버튼 중 하나를 탭하거나, 입력칸을 탭해서 직접 숫자 입력
4. **발주** 버튼 탭
5. "발주 요청 중..." 토스트 → "**발주 추가 완료**" 토스트 (1~2초 내)

### 6.2 발주가 어디로 가나?

- 구글시트에 **자동으로 한 줄 추가**됩니다
- 추가되는 정보: `날짜시간 / SKU / 상품명 / 수량`
- 시트는 **PC 브라우저로 열어서 확인** (URL은 담당자가 알려준 시트)
- ⚠️ PDA 앱에서는 발주 내역 조회 기능 없음 (시트에서만 확인)

### 6.3 발주 실패 메시지별 대응

| 메시지                      | 원인                           | 대응                              |
| --------------------------- | ------------------------------ | --------------------------------- |
| `발주 실패: 시트 연동 실패` | 미니PC 서버가 구글시트에 못 씀 | 담당자 호출                       |
| `발주 실패 (HTTP 500)`      | 서버 내부 오류                 | 1분 후 다시 시도 → 안 되면 담당자 |
| `발주 요청 실패: 연결 오류` | WiFi 끊김                      | WiFi 확인 후 다시 시도            |
| 토스트가 아예 안 뜸         | 앱 멈춤                        | 앱 강제종료 → 재실행              |

## 7. 라벨 인쇄

### 7.1 인쇄 절차

1. 상품 상세 페이지에서 **🏷 라벨** 버튼 탭
2. 수량 입력 다이얼로그 (발주와 비슷):
   - 빠른 선택: **1 / 5 / 10 / 100**
   - 직접 입력: 1~100
3. **인쇄** 버튼 탭
4. "인쇄 요청 중..." 토스트 → 물류PC의 프린터가 라벨 출력
5. "**N장 인쇄 완료**" 토스트가 뜨면 성공

### 7.2 인쇄 실패 메시지별 대응 (중요!)

| 메시지                                            | 원인                                            | 대응                                    |
| ------------------------------------------------- | ----------------------------------------------- | --------------------------------------- |
| `물류PC 응답 없음 (30초 타임아웃)`                | 물류PC의 인쇄 프로그램(`print_agent.py`)이 꺼짐 | **아래 7.3 참고**                       |
| `프린터 오프라인 — 전원/케이블 확인`              | 프린터 자체 문제                                | 프린터 전원 / USB 케이블 확인 후 재시도 |
| `상품명과 라벨이 다름 — 끝XXXXX 템플릿 확인 필요` | 라벨 템플릿 등록 안 됨                          | 담당자 호출                             |
| `라벨 템플릿 없음 (XXXXX.btw)`                    | 신규 상품, BarTender 템플릿 없음                | 담당자 호출                             |

### 7.3 "물류PC 응답 없음" 떴을 때 — 직접 살리는 방법

물류PC에 가서 다음 절차를 따라하세요. (3분이면 끝남)

1. 물류PC 모니터 앞에 앉아서 마우스 흔들어서 화면 깨우기
2. 키보드 **`Win` 키** 누르고 → **`cmd`** 입력 → **엔터**
   - 검은 창(명령 프롬프트)이 뜸
3. 검은 창에 아래 한 줄을 **그대로** 입력하고 엔터:
   ```
   pythonw C:\print_agent.py
   ```
4. 화면에 별다른 메시지 없이 그냥 명령이 끝남 (정상)
5. 검은 창은 **닫지 말고 그냥 최소화**해서 두기 (창 닫으면 같이 꺼질 수 있음)
6. PDA로 돌아와서 라벨 인쇄 다시 시도

#### 만약 위 명령이 "pythonw 인식되지 않습니다" 같은 오류가 나면

```
C:\Python311\pythonw.exe C:\print_agent.py
```

또는

```
python C:\print_agent.py
```

중 하나로 시도. 그래도 안 되면 담당자 호출.

#### PC 재부팅 후에도 자동 실행되게 하려면

명령 프롬프트에서 (관리자 권한이면 더 좋음):

```
reg add "HKCU\Software\Microsoft\Windows\CurrentVersion\Run" /v PrintAgent /t REG_SZ /d "pythonw.exe C:\print_agent.py" /f
```

## 8. 창고 도면 보기

### 8.1 도면 여는 방법

| 방법          | 절차                                              |
| ------------- | ------------------------------------------------- |
| 메인 화면     | 항상 도면이 보임 (별도 동작 불필요)               |
| 상품 상세에서 | **📐 도면** 버튼 → 그 상품 위치가 노란색으로 강조 |

### 8.2 도면 색상 의미

| 색상          | 의미                                     |
| ------------- | ---------------------------------------- |
| 🟡 **노란색** | 현재 검색/조회 중인 상품 위치 (깜빡거림) |
| 🟢 녹색       | 상품이 보관된 일반 셀                    |
| ⬜ 회색       | 비어있는 셀                              |
| 🔲 검정       | 통로 (제품 보관 안 함)                   |
| 🟦 파란색     | 테이블 / 작업대                          |
| 🟥 빨간색     | 물류PC 위치                              |

### 8.3 도면 조작

- **두 손가락으로 벌리기**: 확대 (핀치줌)
- **두 손가락으로 좁히기**: 축소
- **셀 탭**: 그 셀의 상세 페이지로 이동

## 9. 셀 상세 페이지 / 사진 변경

### 9.1 셀 상세 화면

도면에서 셀을 탭하면 진입:

```
┌─────────────────────────────────────────┐
│  [←]                                    │
│                                         │
│                                         │
│         📷 셀 사진 (풀스크린)            │ ← 탭 = 확대 (핀치줌)
│                                         │
│                                         │
│         [A구역 1-1]                     │ ← 셀 위치 라벨
├─────────────────────────────────────────┤
│   [📐 도면]  [라벨]  [✏️ 편집]           │ ← 하단 네비 3버튼
└─────────────────────────────────────────┘
```

- **셀 사진**: 한 셀에 한 장만 등록됨
- **하단 라벨**: 어느 구역의 어느 셀인지 (예: "A구역 1-1")

### 9.2 셀 사진 새로 찍기 / 변경하기

1. 셀 상세에서 **✏️ 편집** 탭
2. "새 사진을 업로드하시겠습니까?" → **확인** 탭
3. 카메라 / 갤러리 선택 메뉴:
   - **카메라**: 즉석에서 촬영
   - **갤러리**: PDA에 저장된 사진에서 선택
4. 촬영/선택 후 자동 업로드 시작
5. "**업로드 완료**" 토스트 → 셀 사진이 즉시 갱신됨

### 9.3 사진 촬영 팁

- 셀 안의 상품들이 **한 화면에 다 보이도록** 조금 떨어져서 촬영
- 어두우면 PDA 플래시 켜기 (카메라 화면 좌측 ⚡ 아이콘)
- 너무 어두우면 셀 라벨이 안 읽혀서 다음에 상품 찾기 어려움

### 9.4 사진 업로드 실패 시

| 증상                 | 대응                                                                                   |
| -------------------- | -------------------------------------------------------------------------------------- |
| "업로드 실패" 토스트 | 다시 한 번 편집 → 사진 변경 시도                                                       |
| 계속 실패            | WiFi 신호 확인 (PDA 우측 위 WiFi 아이콘 막대)                                          |
| 사진이 90도 돌아감   | 알려진 이슈, 다음 업데이트에서 수정. 임시: 갤러리에서 회전시켜 저장 후 갤러리로 업로드 |
| 모든 게 실패         | 담당자 호출                                                                            |

## 10. 박스(QR) 스캔

### 10.1 박스 QR 스캔 절차

1. 외박스에 붙은 **QR 코드**를 PDA 정면을 향하게
2. 트리거 버튼 누름 → "삑" 소리
3. 박스 상세 페이지로 자동 이동

또는 검색창에 박스 번호 직접 입력:

- `BOX-1234` 같은 형식

### 10.2 박스 상세 화면

- **상단**: 박스에 담긴 상품명, 보관 위치
- **중앙**: 박스 사진 + 도면 (박스 위치 노란색 강조)
- **하단 외부 링크 버튼**:
  - **네이버**: 네이버 상품 페이지 (스마트스토어)
  - **쿠팡**: 쿠팡 상품 페이지
  - **1688**: 중국 1688 구매처
  - **FLOW**: FLOW 링크
  - (해당 링크가 없는 항목은 회색으로 비활성)

## 11. 앱 업데이트

### 11.1 자동 업데이트 (기본)

앱 실행 시 새 버전이 있으면 자동으로 다이얼로그가 뜹니다:

## 12. 자주 묻는 질문 (FAQ)

### Q1. 바코드 스캔이 안 돼요

- 트리거 버튼 끝까지 눌렀는지 (PDA 옆면 노란색)
- 빨간 빛 나오는지 (안 나오면 PDA 재부팅)
- 바코드와 거리 10~20cm
- 바코드 구겨짐/오염 확인
- 검색창에 바코드 번호 직접 입력으로 우회

### Q2. "서버 연결 실패" 떠요

- 화면 위 상태바 아래로 쓸어서 WiFi 확인
- 창고 내부 WiFi가 아니면 → 그쪽으로 변경
- WiFi 맞는데 실패 → 톱니바퀴 → 서버 URL 재확인 (`http://100.125.17.60:8000`)
- "연결 테스트" 실패 → 미니PC 자체가 꺼졌을 수 있음 → 담당자 호출

### Q3. 상품을 찾을 수 없다고 떠요

- 상품명 일부만 다르게 입력 (예: "케이스" → "아이폰 케이스")
- SKU나 바코드 번호로 검색 시도
- 신규 상품일 수 있음 → 담당자에게 등록 요청

### Q4. 라벨이 안 찍혀요 — "물류PC 응답 없음"

- **7.3 절차** 그대로 수행 (cmd 열고 `pythonw C:\print_agent.py` 실행)
- 그래도 안 되면 프린터 전원/USB 확인
- 그래도 안 되면 담당자 호출

### Q5. 발주 버튼 눌렀는데 시트에 안 들어가요

- "발주 추가 완료" 토스트 떴는지 먼저 확인
- 시트 새로고침 후 다시 확인 (1~2초 지연 가능)
- "시트 연동 실패" 떴으면 → 담당자 호출

### Q6. 셀 사진이 옛날 사진으로 보여요

- 다른 PDA에서 변경된 경우 최대 1분 지연될 수 있음
- 뒤로가기 → 다시 셀 진입하면 즉시 갱신됨

### Q7. 사진이 90도 돌아서 업로드돼요

- 알려진 이슈, 다음 업데이트에서 수정 예정
- 임시: 갤러리 앱에서 회전시켜 저장 후 갤러리로 업로드

### Q8. 앱이 멈추거나 느려요

- 최근 앱 목록 (▢ 버튼) → 앱 위로 쓸어서 종료 → 재실행
- PDA 재부팅 (전원 길게 → 다시 시작)
- 새 업데이트 있는지 확인 (메인 다시 실행하면 자동 알림)

### Q9. PDA 화면이 너무 빨리 꺼져요

- PDA 시스템 설정 → **디스플레이** → **화면 자동 꺼짐 시간** 늘리기
- (앱에서는 제어 안 함, PDA 시스템 설정에서만 조정)

### Q10. 다른 사람도 같은 PDA 써도 되나요?

- 네, 누구나 사용 가능 (계정 개념 없음)
- 단 발주/사진 변경한 사람이 누구인지 기록 안 됨
- 만약 누가 한 건지 기록이 필요하면 담당자에게 요청

### Q11. WiFi가 자꾸 끊겨요

- 창고 안에서 신호가 약한 곳일 수 있음 (구역 끝, 컨테이너 안 등)
- 신호 좋은 곳으로 이동해서 작업
- WiFi 공유기 위치는 담당자에게 확인

### Q12. 도면이 안 보이거나 깨져 보여요

- 한 번 뒤로가기 → 다시 진입
- 그래도 안 되면 앱 강제종료 → 재실행
- 도면 자체가 잘못 그려졌으면 담당자에게 도면 재설정 요청

## 13. ⚡ 응급 복구 — 모든 게 멈췄을 때

문제 상황별로 **위에서부터 순서대로** 시도하세요.

### 13.1 PDA 한 대만 안 됨

1. **앱 강제종료 → 재실행**
   - 최근 앱 (▢) → 위로 쓸기 → 다시 실행
2. **WiFi 끄기 → 5초 → 다시 켜기**
   - 위 상태바 아래로 쓸기 → WiFi 아이콘 두 번 탭
3. **PDA 재부팅**
   - 전원 길게 → "다시 시작"
4. **다른 PDA로 동일 작업 시도**
   - 다른 PDA에서 정상이면 → 이 PDA만의 문제 → 담당자에게 PDA 점검 요청

### 13.2 모든 PDA가 안 됨 (서버 다운 가능성)

1. **다른 직원 PDA도 확인** — 모두 동일 증상이면 서버 문제
2. **미니PC 모니터 확인**
   - 화면에 빨간 에러 메시지가 떠 있나?
   - 인터넷 케이블이 빠져 있나? (LAN 포트 노란/초록 LED 깜빡이는지)
3. **미니PC 재부팅** (담당자 동의 후)
   - 시작 메뉴 → 전원 → 다시 시작
   - 약 2~3분 후 PDA에서 다시 시도
4. **그래도 안 되면 담당자 즉시 호출**

### 13.3 라벨만 안 찍힘 (스캔/검색은 됨)

1. **물류PC가 켜져 있는지** 확인
2. **프린터 전원/USB** 확인
3. **물류PC에서 인쇄 에이전트 실행** (위 7.3 절차)
4. PDA에서 라벨 다시 인쇄
5. 그래도 안 되면 → 라벨 템플릿(`*.btw`) 누락일 수 있음 → 담당자

### 13.4 발주만 안 됨 (라벨/스캔은 됨)

1. **WiFi 신호** 확인
2. **잠시 후(1분) 다시 시도** (구글시트 일시 지연 가능)
3. **PC 브라우저로 시트 직접 열어보기** — 시트 자체가 안 열리면 구글 측 문제
4. 그래도 안 되면 담당자 호출

### 13.5 도면이 이상하게 보임

1. 앱 강제종료 → 재실행
2. 톱니바퀴 → 메인 화면 다시 진입
3. 그래도 이상하면 → 미니PC에서 도면 데이터 손상 가능 → 담당자
