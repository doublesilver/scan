# 물류창고 PDA 스캐너 앱 사용 가이드

대상: 창고 현장 직원 (IT 비전공)
버전: v5.4.7 기준

---

## 1. 시작하기

### 1.1 앱 실행

- PDA 홈 화면에서 **물류창고 스캐너** 앱 아이콘을 탭
- 자동으로 서버 연결 확인 (3초 정도) → 검색 화면 진입

### 1.2 서버 연결 안 될 때

- "재시도" 버튼 탭
- 그래도 안 되면 → **9번 자주 묻는 질문** 참고

### 1.3 서버 주소 변경 (담당자 안내 시에만)

1. 검색 화면 우측 상단 **톱니바퀴** 아이콘 탭
2. 서버 URL 입력 → **저장**
3. **연결 테스트** 눌러 "연결 성공" 확인

---

## 2. 바코드 스캔

1. 상품의 바코드를 PDA 정면을 향하게 한다
2. PDA 옆면 **노란색 트리거 버튼** 누름
3. "삑" 소리 → 0.5초 내 상품 정보 표시
4. 결과 화면에서 위치/이미지/바코드 확인

### 스캔이 안 될 때

- 바코드가 구겨졌거나 빛 반사가 심한지 확인
- PDA 카메라 렌즈 닦기
- 검색창에 바코드 번호 직접 입력으로 우회

---

## 3. 텍스트 검색

- 검색창에 입력:
  - **상품명** 일부 (예: "케이스")
  - **바코드 번호** (8~13자리)
  - **SKU 번호**
- 결과 카드 탭 → 상세 페이지로 이동

---

## 4. 상품 상세 페이지

상품을 스캔하거나 검색 결과를 탭하면 진입.

### 상단

- 상품명, SKU, 카테고리, 바코드
- 바코드를 **길게 누르면** 클립보드에 복사됨

### 중앙

- 상품 이미지
- 이미지를 탭하면 **썸네일 ↔ 실사 사진** 전환

### 하단 네비바 (3버튼)

- **도면**: 창고 도면에서 이 상품 위치 표시
- **발주**: 구글시트로 발주 정보 전송 → **5번 참고**
- **라벨**: 바코드 라벨 인쇄 → **6번 참고**

---

## 5. 발주 (구글시트 자동 등록)

### 5.1 발주 보내기

1. 상품 상세에서 **발주** 버튼 탭
2. 수량 입력 창이 뜸
   - 빠른 선택: **1 / 10 / 50 / 100**
   - 또는 직접 입력 (1~999)
3. **발주** 탭 → "발주 요청 중..." → "✅ 발주 추가 완료" 토스트

### 5.2 발주 내역 확인

- 발주는 **구글시트에 자동 한 줄 추가**됨
- 시트는 PC 브라우저로 직접 열어서 확인
- (PDA 앱에서 발주 내역 조회는 지원하지 않음)

### 5.3 발주 실패 시

- "발주 실패: 시트 연동 실패" → 담당자 호출
- 네트워크 끊김 / 서버 다운 가능성

---

## 6. 라벨 인쇄

### 6.1 인쇄 시작

1. 상품 상세에서 **라벨** 버튼 탭
2. 수량 선택 (1 / 5 / 10 / 100 또는 직접 입력 1~100)
3. **인쇄** 탭

### 6.2 인쇄 실패 시

- "물류PC 응답 없음" → 물류PC에 인쇄 에이전트(`print_agent.py`)가 꺼졌을 가능성
  - 담당자에게 "프린트 에이전트 죽었어요" 한 마디로 전달
- "프린터 오프라인" → 프린터 전원/USB 케이블 확인
- "라벨 템플릿 없음" → 새 상품이라 템플릿 등록 필요, 담당자 호출

---

## 7. 창고 도면 / 셀 상세

### 7.1 도면 보기

- **방법 1**: 상품 상세 → **도면** 버튼
- **방법 2**: 메인 화면에서 도면이 바로 보임

### 7.2 도면 화면

- 구역(A/B/C 등)별 격자 표시
- 색상:
  - **노란색**: 현재 검색 중인 상품 위치
  - **녹색/회색**: 일반 셀 (상품 있음/없음)
  - **테이블/통로/PC**: 별도 표시
- 두 손가락 벌리면 **확대(핀치줌)** 가능

### 7.3 셀 탭하면

- 해당 셀의 상세 화면으로 이동
- **셀 사진 한 장**이 풀스크린으로 표시됨
- 하단에 셀 라벨 (예: "A구역 1-1")

### 7.4 셀 상세 하단 네비바 (3버튼)

- **도면**: 도면으로 돌아가기
- **(라벨)**: 현재 셀 위치
- **편집**: 사진 변경

---

## 8. 셀 사진 변경

1. 셀 상세에서 **편집** 탭
2. "새 사진을 업로드하시겠습니까?" → **확인**
3. 카메라 / 갤러리 선택
4. 촬영 또는 사진 선택 → 자동 업로드
5. "업로드 완료" 토스트 → 셀 사진 즉시 갱신

### 사진 업로드 실패 시

- WiFi 신호 확인
- 다시 시도 (한 번 더 편집 → 사진 변경)
- 계속 실패하면 담당자 호출

---

## 9. 외박스(박스) QR 스캔

### 9.1 박스 QR 스캔

- 메인 검색 화면에서 박스 QR 코드 스캔
- 또는 검색창에 `BOX-XXXX` 입력

### 9.2 박스 상세

- 박스에 담긴 상품명·위치 표시
- 도면에서 박스 보관 위치 노란색으로 강조
- 외부 링크 버튼: **네이버 / 쿠팡 / 1688 / FLOW** (해당 링크 있을 때만 활성)

---

## 10. 앱 업데이트

### 10.1 자동 업데이트

- 앱 실행 시 새 버전 있으면 **업데이트 다이얼로그** 자동 표시
- **업데이트** 탭 → 자동 다운로드 → **설치** 탭 → 완료
- 설치 후 앱 재시작

### 10.2 업데이트 알림이 안 뜰 때

- 앱을 강제종료(최근 앱 → 위로 스와이프) 후 재실행
- 설정 화면에서 현재 앱 버전 확인 가능

---

## 11. 자주 묻는 질문 (FAQ)

### Q1. 바코드 스캔이 안 돼요

- 트리거 버튼 정확히 눌렀는지 (옆면 노란색)
- 바코드 손상/구김 확인
- PDA 카메라 렌즈 닦기
- 그래도 안 되면 검색창 직접 입력
- 모든 게 안 되면 → **PDA 재부팅** (전원 길게 누르기 → 다시 시작)

### Q2. "서버 연결 실패" 떠요

- WiFi 연결 확인 (창고 내부 WiFi 잡혔는지)
- 화면 우측 상단 톱니바퀴 → 서버 URL 확인
- **연결 테스트** 버튼 눌러봐도 실패 → 담당자 호출

### Q3. 상품을 찾을 수 없다고 떠요

- 다른 검색어(상품명/SKU)로 다시 시도
- 신규 상품일 수 있음 → 담당자에게 등록 요청

### Q4. 라벨이 안 찍혀요 ("물류PC 응답 없음")

- 프린터 전원/USB 확인 후 다시 인쇄 시도
- 그래도 안 되면 → 담당자에게 "프린트 에이전트 죽음" 전달

### Q5. 발주 버튼 눌렀는데 시트에 안 들어가요

- "발주 실패" 메시지 떴는지 확인
- 안 떴으면 시트 새로고침 후 다시 확인 (가끔 1~2초 지연)
- "시트 연동 실패" 떴으면 → 담당자 호출

### Q6. 셀 사진이 옛날 사진으로 보여요

- 다른 PDA에서 변경된 경우 최대 1분 지연 가능
- 뒤로가기 → 다시 셀 진입하면 즉시 갱신

### Q7. 사진이 90도 돌아서 업로드돼요

- 알려진 이슈, 다음 업데이트에서 수정 예정
- 임시 해결: 갤러리에서 회전시켜 저장 후 갤러리로 업로드

### Q8. 앱이 멈추거나 느려요

- 최근 앱 목록 → 앱 위로 스와이프해서 종료 → 재실행
- PDA 재부팅 (전원 길게)
- 새 업데이트 있는지 확인

### Q9. PDA 화면이 자동으로 꺼져서 불편해요

- PDA 시스템 설정 → 디스플레이 → 화면 자동 꺼짐 시간 늘리기
- (앱에서는 제어 안 함)

### Q10. 다른 사람도 같은 PDA로 쓸 수 있나요?

- 네, 계정 개념 없이 누구나 사용 가능
- 단, 발주/사진 변경은 **누가 한 건지 기록 안 됨** (필요하면 담당자에게 요청)

---

## 12. 응급 복구 — 모든 게 멈췄을 때

순서대로 시도:

1. **앱 강제종료 → 재실행**
2. **WiFi 끄기 → 5초 → 다시 켜기**
3. **PDA 재부팅** (전원 버튼 길게 → 재시작)
4. **다른 PDA로 같은 작업 시도** → 한 대만 문제면 그 PDA 이슈
5. **모든 PDA가 안 되면** → 서버 다운 가능성 → 담당자 즉시 호출
