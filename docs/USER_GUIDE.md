# 물류창고 PDA 스캐너 앱 사용 가이드

대상: 창고 현장 직원 (IT 비전공)

---

## 1. 앱 설치 및 초기 설정

### 1.1 앱 설치

1. PDA 홈 화면에서 **파일 관리자** 또는 **Gallery** 앱을 연다
2. 서버에서 다운로드한 APK 파일(`warehouse-scanner-*.apk`)을 찾아서 터치한다
3. "설치" 버튼을 누르고 완료될 때까지 기다린다
4. 앱 사용 권한 요청이 나타나면 **모두 허용** 또는 **동의**를 누른다

### 1.2 앱 초기 실행

1. 앱을 열면 **연결 중** 화면이 나타난다
   - 서버 연결 상태를 확인하는 과정이다 (약 3초)

2. **연결 성공** 시
   - 자동으로 검색 화면으로 넘어간다

3. **연결 실패** 시
   - "재시도" 버튼을 누르면 다시 서버에 연결을 시도한다
   - 이전에 사용했던 상품 정보가 저장되어 있으면 오프라인 모드로 자동 전환된다

### 1.3 서버 URL 설정 (네트워크 변경 시만)

1. 검색 화면에서 우측 상단 **톱니바퀴 아이콘(설정)**을 누른다
2. "서버 URL" 입력칸에 주어진 서버 주소를 입력한다
   - 예: `http://100.125.17.60:8000`
3. **저장** 버튼을 누른다
4. **연결 테스트** 버튼을 눌러 연결 여부를 확인한다
   - 초록색 "연결 성공" 메시지가 나타나면 정상이다

## 2. 바코드 스캔으로 상품 조회

### 2.1 스캔 준비

1. 검색 화면이 열려있는지 확인한다
2. PDA 옆면의 **바코드 스캔 버튼(하드웨어 트리거)**을 누르면 준비 완료
   - 화면에 "PDA 버튼을 눌러 스캔하세요" 안내가 표시된다

### 2.2 바코드 스캔

1. 상품의 바코드가 보이도록 카메라를 향한다
2. PDA 옆면의 **스캔 버튼**을 누른다
   - 스캔 소리가 들리면 성공이다
3. 스캔 완료 시 상품 정보가 자동으로 표시된다

### 2.3 스캔 실패 시

- **다시 스캔**: 바코드가 잘 보이는지 확인하고 스캔을 다시 시도한다
- **검색으로 전환**: 검색창에 바코드 번호를 직접 입력하거나 상품명을 검색한다

---

## 3. 텍스트 검색

### 3.1 검색창 사용

1. 검색 화면의 입력창에 손가락으로 터치한다
2. 키보드가 나타나면 검색어를 입력한다
   - **바코드 번호**: 8~13자리 숫자 입력 시 자동으로 바코드 검색 실행
   - **상품명**: 상품 이름의 일부만 입력해도 된다 (예: "케이스")
   - **SKU ID**: 쿠팡 SKU 번호로도 검색 가능

### 3.2 검색 결과 확인

1. 검색 완료 시 상품 목록(카드 형태)이 표시된다
2. 원하는 상품을 터치하면 상세 페이지로 이동한다
3. 검색을 다시 하려면 입력창의 X 버튼을 눌러 목록을 삭제한다

---

## 4. 상품 상세 페이지

### 4.1 상세 페이지 구성

상품을 스캔하거나 검색 결과에서 선택하면 상세 페이지가 나타난다.

**상단 영역**

- **상품명**: 상품의 한글 이름
- **SKU ID**: 쿠팡 고유 번호
- **카테고리**: 상품 분류 (예: 아이폰 케이스)
- **바코드**: 상품의 바코드 번호 (끝 5자리는 진하게 표시됨)

**중앙 영역**

- **상품 이미지**: 썸네일 또는 실사 이미지가 표시됨
- **이미지 칩**: 현재 보이는 이미지 종류 표시 (썸네일/실사)
  - 이미지를 터치하면 썸네일 ↔ 실사 이미지가 전환된다

**하단 버튼**

- **지도 보기**: 창고 도면에서 이 상품의 위치를 확인한다
- **발주**: 쿠팡 발주 페이지로 이동한다 (준비 중)
- **라벨 인쇄**: 바코드 라벨을 프린터로 인쇄한다

### 4.2 바코드 복사

1. 바코드 텍스트 영역을 **길게 누른다 (약 1초)**
2. "바코드가 복사되었습니다" 메시지가 나타난다
3. 다른 앱에서 바코드를 붙여넣을 수 있다

## 5. 장바구니 추가

### 5.1 장바구니 추가하기

1. 검색 화면에서 상품을 스캔하거나 검색한다
2. 상품 정보가 표시되는 영역의 하단에 **"장바구니 추가"** 버튼이 있다
3. 이 버튼을 누르면 상품이 서버의 임시 장바구니에 저장된다
4. "장바구니 추가 완료" 메시지가 나타나면 성공이다

### 5.2 장바구니 확인 (서버 관리자용)

- 최종 장바구니 내용은 관리자 웹 페이지에서 확인한다
- PDA 앱에서는 임시 저장만 지원한다

## 6. 라벨 인쇄

### 6.1 인쇄 시작

1. 상세 페이지에서 **"라벨 인쇄"** 버튼을 누른다
2. 수량 선택 창이 나타난다

### 6.2 수량 입력

**빠른 선택 버튼**

- **1**: 1장 인쇄
- **5**: 5장 인쇄
- **10**: 10장 인쇄
- **100**: 100장 인쇄

**직접 입력**

- 입력칸에 원하는 수량을 타이핑할 수 있다 (1~100 범위)

### 6.3 인쇄 실행

1. 수량을 선택하거나 입력한다
2. **"인쇄"** 버튼을 누른다
3. 서버에서 인쇄 명령을 받고 프린터에서 라벨이 출력된다
4. "인쇄 요청 완료" 메시지가 나타나면 성공이다

**인쇄 실패 시**

- 프린터 전원이 켜져있는지 확인한다
- 서버와 프린터 연결 상태를 확인한다
- IT 담당자에게 연락한다

## 7. 창고 도면 보기 및 위치 확인

### 7.1 도면 열기

**방법 1: 상세 페이지에서**

1. 상품 상세 페이지의 **"지도 보기"** 버튼을 누른다

**방법 2: 설정 화면에서**

1. 검색 화면 우측 상단 톱니바퀴 아이콘을 누른다
2. **"지도 보기"** 버튼을 누른다

### 7.2 도면 사용법

**도면 화면 구성**

- **좌측**: 층(1층, 2층 등) 선택 (슬라이드로 이동)
- **우측**: 구역(A, B, C 등) 선택
- **중앙**: 선반 격자
  - 초록색 칸: 상품이 보관된 위치
  - 노란색 칸: 현재 검색 중인 상품의 위치
  - 회색 칸: 비어있는 위치

### 7.3 셀(선반) 상세 보기

1. 도면에서 원하는 칸을 **터치**한다
2. 그 선반의 상세 페이지(셀 상세)로 이동한다

---

## 8. 셀(선반) 상세 — 조회 및 편집 모드

### 8.1 셀 상세 페이지 구성

선반(셀) 칸을 터치하면 셀 상세 페이지가 열린다.

**상단**

- **셀 제목**: 몇 층 몇 구역 몇 번째 선반인지 표시
  - 예: "3층 A구역 — A-1-5"

**중앙 - 층별 정보**

- 각 층(하단, 중단, 상단)의 정보 카드가 표시된다
- 현재 조회 모드에서는 이미 저장된 정보만 보인다

**하단 버튼**

- **편집**: 선반 정보를 수정할 수 있다
- **도면으로**: 도면 화면으로 돌아간다

### 8.2 조회 모드 (읽기 전용)

선반에 저장된 정보를 확인하는 모드이다.

**각 층 카드에 표시되는 정보**

- **층 라벨**: 하단/중단/상단 등 선반 이름
- **상품명**: 그 층에 보관된 상품 이름
- **SKU**: 상품의 SKU ID
- **사진**: 촬영된 사진이 있으면 표시됨

### 8.3 편집 모드

1. 하단의 **"편집"** 버튼을 누른다
2. 화면 상단에 "편집 중" 표시가 나타난다
3. 이제 각 층의 정보를 수정할 수 있다

**편집 모드 버튼**

- **+ 층 추가**: 새로운 층을 추가한다 (최대 3층까지)
- **- 층 삭제**: 가장 위의 층을 삭제한다
- **완료**: 모든 변경사항을 저장하고 조회 모드로 돌아간다

## 9. 셀에 상품 등록

### 9.1 층 정보 편집 대화상자 열기

편집 모드에서 각 층의 카드에 있는 **"연필 아이콘(편집)"** 버튼을 누른다.

### 9.2 상품 정보 입력 — 바코드 스캔

**바코드 스캔으로 자동 매칭**

1. 편집 대화상자가 열려있는 상태에서 PDA 옆면의 **스캔 버튼**을 누른다
2. 상품의 바코드를 스캔한다
3. 자동으로 아래 정보가 채워진다:
   - 상품명
   - SKU ID
   - 상품 이미지 (있으면 표시)
   - "✓ 매칭됨" 표시가 나타난다

### 9.3 상품 정보 입력 — 검색으로 매칭

바코드가 없거나 스캔 실패 시 검색으로 상품을 찾을 수 있다.

1. 편집 대화상자의 **"상품 검색"** 버튼을 누른다
2. 검색창이 나타나면:
   - **바코드 검색**: 8~13자리 숫자 입력
   - **상품명 검색**: 상품 이름 입력
   - **SKU 검색**: SKU 번호 입력
3. **검색** 버튼을 누른다
4. 검색 결과 목록에서 원하는 상품을 선택한다
5. 자동으로 상품명과 SKU가 채워진다

### 9.4 상품명 직접 입력

스캔이나 검색 없이 상품명을 직접 입력할 수도 있다.

1. "상품명" 입력칸을 터치한다
2. 키보드로 상품명을 입력한다
3. 완료 후 **"저장"** 버튼을 누른다

**주의**: 이 경우 SKU ID 없이 저장되므로 나중에 발주나 위치 추적이 어려울 수 있다.

## 10. 사진 촬영 및 업로드

### 10.1 사진 촬영

셀 편집 대화상자에서 사진을 추가할 수 있다.

1. **"카메라"** 버튼을 누르면 카메라 앱이 열린다
   - 또는 **"갤러리"** 버튼으로 기존 사진을 선택할 수 있다

2. 선반의 상품이 잘 보이도록 사진을 촬영한다

3. 촬영을 완료하면 자동으로 업로드가 시작된다
   - 진행 상황을 보여주는 로딩 아이콘이 나타난다

### 10.2 업로드 완료 확인

사진 위에 **체크 표시(✓)** 또는 **초록색 체크마크**가 나타나면 업로드 완료이다.

**업로드 실패 시**

- "업로드 실패" 메시지가 나타난다
- **"갤러리"** 버튼으로 다시 시도한다
- 여전히 실패하면 네트워크 연결을 확인한다

### 10.3 사진 삭제

1. 편집 대화상자에서 사진 아래의 **"사진 삭제"** 버튼을 누른다
2. 확인 창이 나타나면 **"삭제"**를 누른다
3. 서버에서 사진이 제거된다

## 11. 외박스(박스) QR 스캔

### 11.1 박스 QR 코드 스캔

1. 검색 화면에서 외박스의 QR 코드를 스캔한다
   - 또는 검색창에 `BOX-`로 시작하는 박스 ID를 입력한다

2. 박스 상세 페이지가 열린다

### 11.2 박스 상세 정보

박스 페이지에는 다음 정보가 표시된다:

**상단**

- 박스에 포함된 상품명
- 구역/선반 태그 (위치 정보)

**중앙 - 도면**

- 현재 박스가 보관된 위치가 **노란색**으로 강조 표시된다
- 도면을 터치하면 상세한 창고 도면으로 이동할 수 있다

**중앙 - 상품 이미지**

- 한국 옵션 이미지 (상품 사진)

**하단 버튼**

- **네이버**: 네이버 상품 페이지로 이동
- **쿠팡**: 쿠팡 상품 페이지로 이동
- **1688**: 1688 구매처로 이동
- **FLOW**: FLOW 링크로 이동
  (링크가 없으면 버튼이 비활성화됨)

## 12. 앱 업데이트

### 12.1 자동 업데이트 알림

앱을 실행할 때마다 새로운 버전이 있는지 자동으로 확인한다.

**업데이트 팝업이 나타나면**

1. **"업데이트"** 버튼을 누른다
2. 새 버전의 APK 파일이 자동으로 다운로드된다
3. 다운로드 완료 후 설치 확인 창이 나타나면 **"설치"**를 누른다

### 12.2 수동 업데이트

1. 서버에서 새 APK 파일을 다운로드한다
2. PDA의 파일 관리자에서 다운로드 폴더를 열고 APK 파일을 찾는다
3. 파일을 터치하여 설치한다

### 12.3 업데이트 후 재시작

앱이 자동으로 재시작되거나 수동으로 다시 열면 새 버전이 활성화된다.

## 13. 자주 묻는 질문 (FAQ)

### Q1. 바코드 스캔이 되지 않습니다

**A** 다음을 확인하세요:

- PDA 옆면 스캔 버튼을 눌렀는가?
- 바코드가 빛에 잘 보이는가?
- PDA 카메라가 깨끗한가? (렌즈를 닦아보세요)
- 바코드가 손상되거나 구겨져있지 않은가?
- 여전히 안 되면 검색창에 바코드 번호를 직접 입력해보세요

### Q2. 상품을 찾을 수 없습니다

**A** 다음을 확인하세요:

- 올바른 바코드를 스캔했는가?
- 상품명 검색으로 다시 시도해보세요
- 상품이 데이터베이스에 등록되어있지 않을 수 있습니다
- IT 담당자에게 상품 등록을 요청하세요

### Q3. 서버에 연결할 수 없습니다

**A** 다음을 확인하세요:

- WiFi가 켜져있고 창고 네트워크에 연결되어있는가?
- 설정에서 서버 URL이 올바르게 입력되어있는가?
- 네트워크 관리자에게 WiFi 연결 상태를 확인하세요
- 재부팅해보세요 (PDA 전원을 껐다 켜기)

### Q4. 라벨이 프린터에서 나오지 않습니다

**A** 다음을 확인하세요:

- 프린터 전원이 켜져있는가?
- 프린터에 라벨지가 남아있는가?
- 프린터와 서버가 같은 네트워크에 연결되어있는가?
- IT 담당자에게 프린터 상태를 확인하세요

### Q5. 인쇄했는데 상품명이 잘못되었습니다

**A** 다음을 하세요:

- 라벨을 폐기합니다
- PDA에서 정확한 상품을 다시 스캔합니다
- 상세 페이지에서 상품명을 확인하고 다시 라벨을 인쇄합니다

### Q6. 오프라인 모드에서는 어떤 기능을 사용할 수 있나요?

**A** 오프라인 모드에서 사용 가능한 기능:

- 이전에 스캔한 상품 조회
- 텍스트 검색 (캐시된 상품만)
- 상품 상세 정보 보기
- 이미지 보기

**사용할 수 없는 기능**:

- 새로운 상품 검색
- 장바구니 추가
- 라벨 인쇄
- 도면 보기
- 셀 정보 편집

### Q7. 사진이 업로드되지 않습니다

**A** 다음을 확인하세요:

- 네트워크 연결이 안정적인가?
- 사진 용량이 너무 크지 않은가? (재촬영해보세요)
- 로딩 아이콘이 계속 나타나면 완료될 때까지 기다리세요
- 여전히 안 되면 갤러리에서 다시 업로드를 시도하세요

### Q8. 장바구니 내용을 확인하려면?

**A** PDA 앱에서는 장바구니 조회 기능이 없습니다.

- 관리자 웹 페이지(http://서버주소:8000/admin)에서 확인하세요
- 또는 IT 담당자에게 문의하세요

### Q9. 앱이 자꾸 멈춥니다

**A** 다음을 시도하세요:

- 앱을 완전히 종료했다 다시 열기
- PDA 재부팅 (전원 껐다 켜기)
- 앱 업데이트가 있는지 확인
- 여전히 멈추면 IT 담당자에게 연락하세요

### Q10. 앱 버전은 어디서 확인하나요?

**A** 설정 화면의 하단에 현재 앱 버전이 표시됩니다.

- 예: "v1.0.0"
