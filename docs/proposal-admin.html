A안과의 차이점

A안 (스트랩 전용)

  • 스트랩 제품·기종·재질 추가는 자유롭게 가능
  • 스트랩 외 상품군(케이스·게임 악세서리 등) 추가 시 별도 개발 필요
  • 출시 현황표: 스트랩 × 기종 전용

B안 (확장 가능)

  • 분류: 카테고리별 항목 자유 추가
  • 관리자 화면에서 직접 추가 가능
  • 출시 현황표: 어떤 상품이든 교차표 생성

구조적 차이

A안은 스트랩 전용으로 설계된 시스템입니다.
B안은 상품군 제약 없이 확장 가능한 구조로 설계됩니다.
초기에는 스트랩만 운용하더라도, 향후 게임 악세서리·케이스 등 타 상품군을 추가 개발 없이 수용할 수 있습니다.

각 단계 완료 시 납품 후 직접 확인하신 다음 다음 단계 진행 여부를 결정하시면 됩니다. 완성된 기능부터 순차적으로 실사용에 투입됩니다.
2
단계별 결제
각 단계 착수 시 50%, 납품 완료 후 50%를 결제합니다. 다음 단계 진행 여부는 납품 확인 이후에 결정하시면 됩니다.
3
우선순위 변경 가능
2-1단계 이후, 2-2 / 2-3 / 2-4의 진행 순서는 업무 우선순위에 따라 조정 가능합니다. 협의를 통해 언제든지 순서를 변경할 수 있습니다.
4
중간 중단 가능
원하시는 단계에서 중단하셔도 됩니다. 2-1단계 완료 후 나머지 단계는 시점을 달리해 진행하거나 진행하지 않아도 무방합니다.
